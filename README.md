# BuildMC Launcher Website

Official website for the BuildMC Launcher, built with Astro and deployed on GitHub Pages.

## Pages

- **Home** — BuildMC Launcher landing page with download button
- **Downloads** — Launcher installer, plugins, modpacks, and resource packs
- **Terms** — Nutzungsbedingungen for the Discord app
- **Privacy** — Datenschutzerklärung for the Discord app

## Deployment

The site is deployed automatically via GitHub Actions to GitHub Pages.

## Local Development

```bash
npm install
npm run dev
```

## 🧞 Commands

| Command                 | Action                                      |
| :---------------------- | :------------------------------------------ |
| `npm install`           | Installs dependencies                       |
| `npm run dev`           | Starts local dev server at `localhost:4321` |
| `npm run build`         | Build your production site to `./dist/`    |
| `npm run preview`       | Preview your build locally                  |
| `npm run astro ...`     | Run CLI commands like `astro add`           |
